Feelings behind Words

Introduction

Political discourse in social media is crucial for informed democratic decision-making but is susceptible to manipulation through persuasive techniques and misinformation. This project focuses on automating the detection of persuasion techniques employed in social media content using a multimodal Artificial Neural Network (ANN) model.

Goals

The primary goal of this project is to develop a robust model capable of analyzing both textual and image-based content in social media posts, detecting various persuasion techniques such as appeal to authority, repetition, and bandwagon.

Methodology

Model Architecture

Model 1 (Sentiment Analysis): Utilizes BERT for tokenization and TensorFlow for sentiment classification.

Model 2 (Image Classification): Employs Convolutional Neural Networks (CNNs) for image feature extraction and classification.

Combined Model

Integrates outputs from Model 1 and Model 2 using late fusion to enhance accuracy in detecting persuasion techniques.

Data sourced from the 15th International Workshop on Semantic Evaluation (SemEval 2021), comprising 888 labeled social media posts.

Results

Overall Accuracy: The combined model achieves an accuracy of 89.2% in detecting persuasion techniques.

Label-wise Accuracy: Varied accuracies across 20 persuasion techniques, highlighting strengths and limitations in detection.

Analysis

Disparities in accuracy underscore the effectiveness of multimodal approaches in discerning rhetoric within social media content. The model enhances media literacy and aids in minimizing propaganda dissemination.

Conclusion

By providing valuable insights into manipulative content, this project lays the foundation for enhanced awareness and critical engagement with media consumption. Further advancements could include integrating advanced algorithms and contextual analysis for broader societal impacts.
